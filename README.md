# Customer_behaviou.Analysis
Analyzed customer purchasing patterns to identify trends, segments, and key factors influencing buying behavior. Performed data cleaning, EDA, and visualization using Python and SQL to derive actionable business insights.  Tech Stack: Python, Pandas, Matplotlib, Seaborn, SQL,POWER BI




🔄 Steps Performed
Data loading and cleaning using Python
Exploratory Data Analysis (EDA)
SQL queries for deeper insights
Dashboard creation in Power BI
Report/PPT generation
📊 Dashboard

Interactive Power BI dashboard showcasing customer segments, revenue trends, and purchasing patterns.

📈 Results
Identified customer segments (New, Returning, Loyal)
Analyzed spending behavior and discount impact
Derived actionable business insights
How to Run:-




Load dataset in Python
Perform cleaning & EDA
Execute SQL queries
Open Power BI dashboard file
